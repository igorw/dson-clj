# Doge Serialized Object Notation

A parser for the [Doge Serialized Object Notation](http://dogeon.org/)
("dson") format. Using instaparse.

## Usage

    (dson "such \"foo\" is \"bar\" wow")
    ; => {"foo" "bar"}
